# VSI PDF Viewer — Quick Reference Card

## Touch Gestures

| Gesture | Action |
|---|---|
| One finger drag | Scroll / pan the page |
| Two fingers spread apart | Zoom in (anchored to the point between your fingers) |
| Two fingers pinch together | Zoom out (anchored to the point between your fingers) |
| Double-tap | Toggle zoom (normal ↔ 2.5x) |
| Swipe left | Next page |
| Swipe right | Previous page |

## Zoom Levels

The toolbar dropdown offers presets from 25% to 1000%, plus Full Page and Full Width.
Pinch and Ctrl+wheel zoom can go up to the configured maximum (default 400%, configurable up to 1000%).

## Swiping When Zoomed In

1. Scroll to the edge of the page first
2. Then swipe to change pages

Swiping only navigates when you are ALREADY at the edge before the swipe starts.

## Search

1. Type your search term in the search box
2. Tap 🔎 to find and highlight matches
3. Tap ✕ to clear highlights

Search ignores uppercase/lowercase and accents (e.g. "curacao" finds "Curaçao").

## Protected Documents

When you see 🔒 Protected in the toolbar:

- Viewing, zooming, scrolling, and searching still work
- Downloading, saving, printing, and long-press menus are blocked

## Keyboard (Desktop)

| Key | Action |
|---|---|
| ← → | Scroll horizontally or navigate pages |
| ↑ ↓ | Scroll vertically |
| Ctrl + ← | Previous page (instant) |
| Ctrl + → | Next page (instant) |
| Ctrl + mouse wheel | Zoom in / out (anchored to cursor position) |

## Mouse (Desktop)

| Action | Result |
|---|---|
| Click and drag | Pan / scroll the PDF (grab cursor) |
| Ctrl + mouse wheel | Zoom in / out at cursor position |
| Fast horizontal drag at edge | Navigate to next / previous page |
