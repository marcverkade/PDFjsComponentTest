# PDF Viewer — Snelkaart

## Aanraakgebaren

| Gebaar | Actie |
|---|---|
| Eén vinger slepen | Scrollen / pagina verschuiven |
| Twee vingers uit elkaar spreiden | Inzoomen |
| Twee vingers samenknijpen | Uitzoomen |
| Dubbeltikken | Zoom wisselen (normaal ↔ 2,5x) |
| Veeg naar links | Volgende pagina |
| Veeg naar rechts | Vorige pagina |

## Werkbalk

    ⏮ ◀  [ 3 ] / 12  ▶ ⏭  |  −  Zoom: 125%  +

| Knop | Actie |
|---|---|
| ⏮ | Eerste pagina |
| ◀ | Vorige pagina |
| [ 3 ] / 12 | Tik op het nummer, typ een pagina en druk op Enter |
| ▶ | Volgende pagina |
| ⏭ | Laatste pagina |
| − | 25% uitzoomen |
| + | 25% inzoomen |

## Vegen wanneer ingezoomd

1. Scroll eerst naar de rand van de pagina
2. Veeg dan om van pagina te wisselen

Vegen navigeert alleen wanneer u AL bij de rand bent voordat de veeg begint.

## Zoeken

1. Typ uw zoekterm in het zoekveld
2. Tik op 🔎 om overeenkomsten te vinden en te markeren
3. Tik op ✕ om markeringen te wissen

Zoeken negeert hoofdletters/kleine letters en accenten.

## Beveiligde documenten

Wanneer u 🔒 Beveiligd ziet in de werkbalk:

- Bekijken, zoomen, scrollen en zoeken werken nog steeds
- Downloaden, opslaan, afdrukken en lang-indruk-menu's zijn geblokkeerd

## Toetsenbord (Desktop)

| Toets | Actie |
|---|---|
| ← → | Horizontaal scrollen of pagina's navigeren |
| ↑ ↓ | Verticaal scrollen |
| Ctrl + ← | Vorige pagina |
| Ctrl + → | Volgende pagina |
| Ctrl + muiswiel | In- / uitzoomen |
